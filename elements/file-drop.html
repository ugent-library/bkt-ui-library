File Drop

Accessible drag-and-drop file upload zones. The entire area is a clickable label that activates a hidden file input. Drag-over states are managed via JavaScript by toggling the drag-over class.

Basic file drop zone

Default — idle state
Drag-over state — .drag-over class applied

JavaScript integration

Drag and drop handlers
javascript
// Add drag-over class on dragenter/dragover
ondragover="event.preventDefault(); this.classList.add('drag-over')"

// Remove drag-over class on dragleave/dragend
ondragleave="this.classList.remove('drag-over')"

// Handle drop and remove drag-over class
ondrop="event.preventDefault(); this.classList.remove('drag-over'); handleFiles(event.dataTransfer.files)"

Accessibility

  • The entire drop zone is a <label> that activates the hidden file input
  • Screen readers can navigate to and activate the file input directly
  • Include accept attribute to filter file types for better UX
  • Drag and drop is an enhancement — keyboard file selection always works
  • Provide clear instructions about accepted file types and size limits
  • Handle upload errors gracefully with appropriate feedback

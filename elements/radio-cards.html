Radio Cards

Bootstrap-based. Large, touch-friendly radio button alternatives. Used for publication type selection in deposit flows. The hidden radio input drives state; the visible card responds via CSS adjacent sibling selectors.

Accessibility

  • Screen readers announce the selected state and group context

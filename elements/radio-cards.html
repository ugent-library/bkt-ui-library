Radio Cards

Large, touch-friendly radio button alternatives. Used for publication type selection in deposit flows. The hidden radio input drives state; the visible card responds via CSS adjacent sibling selectors.

Accessibility

  • Always wrap radio card groups in <fieldset> with <legend>
  • The legend can be visually hidden with visually-hidden if the context is clear
  • Radio inputs remain focusable and keyboard navigable
  • Screen readers announce the selected state and group context
  • Touch targets are large (minimum 44px) for mobile accessibility

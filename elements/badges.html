Badges

Badges label states, types, and categories. They never initiate actions and they're never decorative. Every badge is an answer to a question someone might have while scanning a list. If nobody would ask the question, the badge shouldn't be there.

Usage rules

✓ Do

  • Use badge text-bg-success and badge text-bg-warning for access status — not generic Bootstrap colours
  • Use text-bg-success for "Published" — the semantic meaning matters
  • Use text-bg-warning or text-bg-danger for states that need attention
  • Keep badge text to 1–3 words

✗ Don't

  • Use text-bg-danger for "Restricted" — restricted isn't an error
  • Stack more than 3 badges in a card header
  • Use colour alone to convey meaning (always include text)
  • Use badges as buttons or links

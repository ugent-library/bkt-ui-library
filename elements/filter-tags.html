Filter Tags

Removable filter indicators. Show active search filters and allow users to remove them. Used in search interfaces to display current filter state.

Accessibility

  • Each filter tag is a <button> for keyboard accessibility
  • Include the filter name and value in the button text
  • The close icon is decorative (aria-hidden="true")
  • Consider adding aria-label for complex filters: aria-label="Remove filter: Type is Journal article"
  • Screen readers announce the full button text when focused
  • Filter removal is confirmed immediately — no confirmation dialogs needed

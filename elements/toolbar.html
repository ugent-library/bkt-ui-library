Accessibility

  • Add role="search" to search toolbars with aria-label="Search and filter research output"
  • Use semantic headings (<h1>) in bc-toolbar__title when appropriate
  • bc-toolbar__item elements are flex items within bc-toolbar__left/bc-toolbar__right/bc-toolbar__middle containers
  • Navigation toolbars should use <nav> in the middle section
  • Screen readers navigate toolbar content as part of the page flow
  • Interactive elements within toolbars remain fully keyboard accessible

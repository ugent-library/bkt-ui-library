Avatars

User avatars display initials or profile images. They appear in author lists, contributor rows, and user profiles. Always include meaningful alt text or aria-label for screen readers.

Default (medium), small, large

Accessibility

  • Avatars are decorative — they do not need alt text
  • The user's name appears in adjacent text content
  • If the avatar is interactive (clickable), add appropriate ARIA labels
  • Initials should be 2 characters maximum for readability

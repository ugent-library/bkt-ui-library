Syntax reference

Basic tokens

Token Example Matches
type: type:journal_article Works of that type
author: author:smith Works by an author with that family name
affiliation: affiliation:sciences Works from that faculty
year: year:2024 Works published that year
status: status:submitted Works in that Biblio lifecycle status
has_file: has_file:true Works with (or without) a file attached
open_access: open_access:true Works with open access full text
identifier: identifier:10.1002/lpor.202401234 Exact DOI, handle, or ISBN

Range and comparison operators

Syntax Example Meaning
field:a..b year:2020..2024 Value between a and b (inclusive)
field:>value year:>2020 Value greater than
field:<value year:<2024 Value less than
field:>=value year:>=2020 Value greater than or equal

Negation and multiple values

Syntax Example Meaning
-field:value -type:dataset Exclude works matching this token
field:a,b type:journal_article,book Match any of these values (OR)
"quoted text" "quantum computing" Exact phrase in title or abstract

Example queries

type:journal_article affiliation:sciences open_access:true year:2020..2025

Open access journal articles from Sciences, 2020–2025

status:submitted -type:dataset updated_since:2024-01-01

Submitted works (excluding datasets) updated since January 2024

identifier:10.1002/lpor.202401234

Look up a record by DOI

affiliation:engineering type:dataset,software has_file:true

Datasets and software from Engineering with files attached

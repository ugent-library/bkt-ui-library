Accessibility statement

Ghent University Library is committed to making its websites accessible, in accordance with the Bestuursdecreet of 7 December 2018 (opens in new window), which transposes European Directive 2016/2102 (opens in new window).

This accessibility statement applies to the websites of the Ghent University Library (Dutch and English versions):

  • Website: https://lib.ugent.be
  • Catalogue: https://libcatalog.ugent.be

About this website

lib.ugent.be consists of two parts with different technical ownership:

Content pages (lib.ugent.be)
Library information, services, opening hours, and contact pages. Built and maintained by Ghent University and Ghent University Library.
Search and catalogue (libcatalog.ugent.be)
The homepage, search functionality, and catalogue are provided by Ex Libris Primo NDE (opens in new window), a third-party discovery platform. We can configure some aspects (labels, colours, page titles) and apply CSS overrides, but we cannot change the underlying application code. Issues in the core application require action by the vendor.

Compliance status

This website partially complies with the Web Content Accessibility Guidelines (WCAG) 2.2 (opens in new window) level AA. The non-compliant areas are listed below.

How we tested

An in-depth internal accessibility audit has been carried out. This audit consisted of a manual and automated analysis of various page types across both domains in both Dutch and English. Download the detailed audit report.

Audits October 2025 — manual audit by Ghent University Library.
April 2026 — manual and automated audit by Ghent University Library.
Scope lib.ugent.be (content pages) and libcatalog.ugent.be (catalogue) tested as a single integrated user experience. Both unauthenticated and authenticated (UGent account) to reach search results and detail pages.
  • lib.ugent.be/nl and /en (homepage — redirects to Primo NDE)
  • lib.ugent.be/nl/libraries and /en/libraries (+ individual library pages including Boekentoren)
  • lib.ugent.be/nl/services and /en/services (+ subpages: registration, loans, scanning, ILL, study locations, etc.)
  • lib.ugent.be/nl/info and /en/info (+ subpages: regulations, research tips, e-resources, open data, etc.)
  • libcatalog.ugent.be search results page
  • libcatalog.ugent.be detail pages
Tools WAVE extension, Axe DevTools, VoiceOver (screen reader), keyboard navigation, browser DevTools.

Non-accessible content

The content listed below is not yet accessible for the following reasons:

Content pages (lib.ugent.be)

Items within our direct technical control:

WCAG Criterion Issue Severity Temporary solution Status
1.4.3 Contrast The cookie consent button (Piwik) has a contrast ratio of 4.27:1. Low The button is identifiable by position and shape; settings are also accessible via browser settings. CSS override correction planned.

Search and catalogue (Ex Libris Primo NDE)

The catalogue is provided by a third-party discovery platform. We can configure certain aspects, but underlying code issues require action by the vendor. View the Ex Libris Primo NDE accessibility information (opens in new window).

WCAG Criterion Issue Severity Temporary solution / Alternative Status
1.4.10 Reflow Content can be enlarged without increasing line length. Requires 400% zoom to work. High Use the browser's "simplified view" or contact the library for assistance. Fix planned by vendor (July 2026).
1.4.4 Resize Text Text can be enlarged. Requires 200% zoom to work. High Use the browser's "simplified view" or contact the library for assistance. Fix planned by vendor (July 2026).
1.3.1 Info & relations Non-meaningful H1 titles ("home", "search", "fulldisplay"). High Use the page title in the browser tab or breadcrumbs for context. Raised with vendor (April 2026).
1.3.1 Info & relations Broken heading hierarchy and empty headings in the DOM. Moderate Navigate via landmarks or screen reader search functions to find sections. Raised in April 2026. Awaiting vendor feedback.
1.3.1 Info & relations Some <nav> elements lack unique aria-labels. Moderate Explore the content of navigation blocks manually to determine function. Raised in April 2026. Awaiting vendor feedback.
2.4.4 Link purpose Links lack descriptive ARIA labels containing work titles. Moderate The work's title is located directly above or next to the action button. Raised in April 2026. Awaiting vendor feedback.

Improvement plan

Recently resolved (April 2026): systemic review performed

  • Addition of a skip link for faster keyboard navigation on lib.ugent.be.
  • Systematic addition of unique aria-labels to navigation regions on lib.ugent.be.
  • Improved focus visibility for all interactive elements via CSS overrides.
  • Clean-up of empty headings and improved heading hierarchy on lib.ugent.be.
  • Correction of the breadcrumb to a semantic list structure.
  • English homepage title now displays English name.

Planned

Short term (3 months)

  • Address the contrast issue in the cookie consent widget.

Medium term (3–6 months)

  • Follow-up on promised updates from software vendor Ex Libris (including the Reflow fix expected in July 2026).
  • We will raise the remaining navigation and heading issues with Ex Libris once the current items are resolved, to avoid overloading the vendor with multiple issues at once.
  • Investigate solutions if no vendor fixes are possible.
  • Establish annual audit cadence; include accessibility in procurement SLAs.

Suggested alternatives in the meantime

If you encounter problems with the visual display or navigation in the catalogue, our library staff will be happy to assist you in retrieving metadata or reserving documents via lib.service@ugent.be or at the desk of our various library locations.

Navbar & Topbar

The bt-navbar component is the persistent application topbar. It appears on every page, sticks to the top, and defines the primary navigation and identity of the surface. Backoffice and public variants differ in content and tone — not in structure.

Backoffice navbar

Carries the application logo, section navigation, role switcher, and account menu. The account avatar doubles as a dropdown trigger.

Standard — researcher logged in

Public navbar

Carries the repository identity, site navigation, and a shortcut to the researcher's own deposit workflow. The site name acts as the <h1> equivalent for identity — it is a heading styled as nav text, not a bare link.

Standard — anonymous visitor

Active link state

Apply aria-current="page" to the link matching the current URL. The CSS responds to this attribute — no extra class needed.

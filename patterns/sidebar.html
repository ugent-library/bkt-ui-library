Sidebar Navigation

bt-sidebar is the persistent section navigator inside u-layout--app. It carries meaningful navigation state and should be used only for real page navigation, not decoration.

bt-sidebar — section navigator

Used in backoffice list and detail pages. Sits in the left column of u-layout--app, sticky below the topbar. Links use Bootstrap .nav-link directly — no extra wrapper class needed beyond the sidebar container.

Layout integration

bt-sidebar is a direct child of u-layout--app and is made sticky by the layout shell. Do not add position: sticky manually.

html — u-layout--app with bt-sidebar
<div class="u-layout--app" data-surface="backoffice">
  <header class="bt-navbar" role="banner">…</header>

  <nav class="bt-sidebar" aria-label="Section navigation">
    <!-- nav links -->
  </nav>

  <main id="main-content">
    <!-- u-main__header → u-main__body … -->
  </main>
</div>

Sidebar Navigation

Two sidebar components serve distinct purposes. bt-sub-sidebar is a persistent section navigator inside u-layout--app. bt-stepper is a step-progress navigator inside u-layout--deposit. Neither is decorative — both carry meaningful navigation state.

bt-sub-sidebar — section navigator

Used in backoffice list and detail pages. Sits in the left column of u-layout--app, sticky below the topbar. Links use Bootstrap .nav-link directly — no extra wrapper class needed beyond the sidebar container.

bt-stepper — deposit step navigator

Used inside u-layout--deposit for multi-step workflows. Each step shows a numbered circle, a label, and an optional "Required" indicator. Completed steps replace the number with a check icon. The current step carries aria-current="step".

Layout integration

bt-sub-sidebar is a direct child of u-layout--app. bt-stepper is a direct child of u-layout--deposit. Both are made sticky by the layout shell — do not add position: sticky manually.

html — u-layout--app with bt-sub-sidebar
<div class="u-layout--app" data-surface="backoffice">
  <header class="bt-navbar" role="banner">…</header>

  <nav class="bt-sub-sidebar" aria-label="Section navigation">
    <!-- nav links -->
  </nav>

  <main id="main-content">
    <!-- bt-toolbar, bt-side-bar, bt-content-area … -->
  </main>
</div>
html — u-layout--deposit with bt-stepper
<div class="u-layout--deposit" data-surface="backoffice">
  <header class="bt-navbar" role="banner">…</header>

  <nav class="bt-stepper" aria-label="Deposit steps">
    <!-- stepper items -->
  </nav>

  <main id="main-content">
    <!-- form sections -->
  </main>
</div>

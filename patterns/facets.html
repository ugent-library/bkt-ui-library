Sidebar with Facets

bt-sub-sidebar + facets

Facet groups use Bootstrap's collapse component and standard form-check inputs inside fieldset/legend groups. The old custom CSS grid (bt-facets, bt-facet-check, bt-facet-name, bt-facet-separator) is removed. See templates/biblio-public/public-search.html for the full live example.

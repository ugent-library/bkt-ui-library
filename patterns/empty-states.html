Empty states

Two kinds of empty state: an inline blank slate (bt-blank-slate) for a card or section with no content, and a guided empty composed from utilities for a full search or results area.

Inline empty

The blank slate (bt-blank-slate) is a small, centered empty for a card or section: no files on a record, an unfilled panel. It carries its own styling (centering, padding, tint). For a full search or results area, use the guided pattern below instead.

Guided empty — search / results area

A zero-results search state is a guided empty state, not the centered bt-blank-slate. It stays left-aligned in the results column, names the query, and offers recovery — remove a filter, broaden or re-spell the terms. Compose it from utilities and give the region role="status" so screen readers announce it. Use bt-blank-slate for small inline empties (no files, empty section); use this for a full results area.

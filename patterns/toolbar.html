Usage notes

  • The bt-toolbar__title is an <h1> when the toolbar heads the page view.
  • bt-toolbar__item siblings inside __left/__right/__middle get automatic spacing.
  • Search toolbars carry role="search" with a distinct aria-label; view switches are a Bootstrap btn-group with input.btn-check — see the live list templates.

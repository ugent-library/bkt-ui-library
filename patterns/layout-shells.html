Layout Shells

Three CSS grid shells handle every page layout in the system. Each shell owns its topbar, optional sidebar, and main content area. They replace the old flex + overflow: hidden approach that broke sticky positioning.

The three shells

All shells use --s-topbar-height (52px, defined in _surfaces.scss) for sticky sidebar offsets. Public <main> uses Bootstrap .container inside it — the shell itself has no max-width.

Topbar height token

--s-topbar-height is set in _surfaces.scss. If the topbar height ever changes, update the token in one place.

scss — _surfaces.scss
/* In _layouts.scss — no fallback needed, token is always defined */
.u-layout--app > .bt-sub-sidebar {
  top: var(--s-topbar-height);
  height: calc(100vh - var(--s-topbar-height));
}

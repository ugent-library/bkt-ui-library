One primary per context: the access CTA. Add to list is btn-secondary on the detail header, btn-ghost on cards; Cite is btn-ghost everywhere. A CTA that navigates (login, external, file) is an <a>, never a <button>.

Every card: Cite (ghost) + Add to list (ghost) + the access CTA. The Add to list picker panel is documented at Panel.

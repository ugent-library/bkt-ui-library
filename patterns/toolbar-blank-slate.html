Toolbar & Blank Slate

The toolbar (bt-toolbar) is the page header pattern used consistently across every backoffice screen. The blank slate (bt-blank-slate) handles empty states. Both are existing components — documented here with their real class names.

bt-toolbar — standard

Title left, actions right — the most common pattern

Research output

html
<div class="bt-toolbar">
  <div class="bt-toolbar__left">
    <h1 class="bt-toolbar__title">Research output</h1>
  </div>
  <div class="bt-toolbar__right">
    <div class="bt-button-toolbar">
      <button class="btn btn-secondary">Import</button>
      <button class="btn btn-primary">Add research output</button>
    </div>
  </div>
</div>

Grid-based layout — real component structure from the existing system

Reality note: The existing bt-facets component uses a CSS grid with three columns: bt-facet-check, bt-facet-name, bt-facet-count as sibling divs inside bt-facets. This differs from what you'd expect from a standard nested list. The grid is defined in your existing CSS. The above demo replicates the visual output using inline styles for portability.
html — existing bt-facets structure
<div class="bt-facets">
  <div class="bt-facet-title">Type</div>

  <div class="bt-facet-check">
    <input class="form-check-input" id="f-journal" type="checkbox" checked>
  </div>
  <div class="bt-facet-name">
    <label for="f-journal">Journal article</label>
  </div>
  <div class="bt-facet-count">1.234</div>

  <div class="bt-facet-separator"></div>
  <div class="bt-facet-title">Year</div>
  <!-- repeat pattern -->
</div>

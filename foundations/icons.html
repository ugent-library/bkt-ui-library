Adding icons

Icons are built from SVG files in assets/icon-font-source/ using Fantasticon. To add a new icon:

  1. Drop your SVG into assets/icon-font-source/
  2. Run npm run build:icons
  3. The font files and _icon-font.scss update automatically — then run npm run build:css or restart npm run dev

SVG requirements for Fantasticon: single-colour, filled paths only, no strokes, ideally on a square viewBox. Fantasticon normalises the viewBox during build.
